# react-phonecat

A React rewrite of the Angular tutorial.

Accompanying blog post: [jgebhardt.github.io/blog/react-phonecat](http://jgebhardt.github.io/blog/react-phonecat)

### Getting set up

Clone the git repository:

```
git clone https://github.com/jgebhardt/react-phonecat.git && cd react-phonecat
```

Install dependencies via npm:

```
npm install
```


Finally, run the server:

```
npm run start
```

react-phonecat is built on top of the [react-quickstart](https://github.com/andreypopp/react-quickstart) project.
